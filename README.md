# MBTI Growth Platform 🌱

당신의 성격유형으로 시작하는 맞춤형 성장 여정

## 🌟 주요 기능

- ✅ 과학적으로 검증된 MBTI 성격 진단
- ✅ 16개 성격유형별 맞춤 성장 전략
- ✅ 커리어, 인간관계, 개인성장 가이드
- ✅ AI 기반 1:1 맞춤 코칭
- ✅ 성장 트래킹 및 분석
- ✅ 커뮤니티 및 멘토링 시스템

## 🚀 배포

### Vercel로 배포하기

1. Vercel CLI 설치
```bash
npm i -g vercel
```

2. 프로젝트 디렉토리에서 배포
```bash
vercel
```

3. 프로덕션 배포
```bash
vercel --prod
```

### 로컬 실행

```bash
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080 접속
```
